# React + TypeScript + Vite + Firebase Auth

## How to Run Locally

1. Clone the repo
2. `cd vite-react-app`
3. `npm i`
4. Fill the environment variables with `touch .env`, following the `.env.example`
5. `npm run dev`

## Features

- Firebase Auth (Sign-up, Sign-in, Logout)
- Change password
- Responsive UI for Dashboard Boilerplate
